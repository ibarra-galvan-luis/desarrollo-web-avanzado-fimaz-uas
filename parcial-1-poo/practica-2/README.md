Practica 2  Herencia en PHP

Explicacion
Se implemento herencia usando una clase base llamada Usuario y una clase derivada llamada Admin.

La clase Admin extiende a Usuario utilizando la palabra clave extends, lo que permite reutilizar los atributos y métodos de la clase base.

Diferencias entre Usuario y Admin

Usuario:
Tiene atributos nombre y correo
Tiene getters y setters

Admin:
Hereda todo de Usuario
Agrega el método getRol()
